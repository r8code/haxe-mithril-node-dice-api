## Dice API Haxe Version (Complete - Definitive Version)
- Mithril-Hx
- HxNodeJS
- Express
- Live-server
- Concurrently
- Haxe-Watchify
- Nodemon

### Build
```
[install haxe libs]
npm install
npm run build 
open http://localhost:3000
```

### Dev
```
[install haxe libs]
npm install
npm run dev
open http://localhost:8080 [live-server]
```
